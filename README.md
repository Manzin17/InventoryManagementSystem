# Inventory Management System

A sleek and efficient Android application designed to help businesses and individuals track, manage, and organize their stock and inventory in real-time. 

---

* **Product Tracking:** Add, edit, and delete inventory items with details like names, quantities, prices, and descriptions.
* **Category Management:** Group products into distinct categories for seamless navigation and filtering.
* **Search & Filter:** Quickly find items using a dynamic search bar or filter by category/stock availability.
* **Clean UI/UX:** Built following modern Android design guidelines for a smooth user experience.
